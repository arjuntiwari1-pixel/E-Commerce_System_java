# E-Commerce System (Java)

## Description
This is a simple Java-based E-Commerce system.  
It allows users to view products, add them to cart, and calculate the total bill.

## Features
- View products
- Add products to cart
- View cart
- Checkout and see total bill

## Tools Used
- Java
- ArrayList
- Scanner

## How to Run
1. Compile: javac ECommerceSystem.java  
2. Run: java ECommerceSystem
